# scrollmap-search-panel

Show search-panel results on the scrollbar.

## Installation

To install `scrollmap-search-panel` run `ppm install asiloisad/pulsar-scrollmap-search-panel` to install a package directly from the GitHub repository.

Layer package for [scrollmap](https://github.com/asiloisad/pulsar-scrollmap).

Requires [search-panel](https://github.com/asiloisad/pulsar-search-panel).
